如何安装？

@alife/rdx#

Rdx 是一个相当轻量的状态管理框架。

通过 graph 的数据结构来存储动态收集的状态节点的依赖关系，从而更好的调度控制组件的渲染，处理协同同步和异步事件流。

参考了Recoil的部分设计理念。

安装#

npm install @alife/rdx 或者 yarn add @alife/rdx
