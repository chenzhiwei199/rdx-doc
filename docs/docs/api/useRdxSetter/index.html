useRdxSetter(state)

会根据当前参数的 RdxState 来返回设置器，通过这个设置器，可以修改相应的 RdxState 的值。

function useRdxState<GModel>(state: RdxState<GModel>): TNext<GModel>;
type TNext<GModel> = (value: GModel | ((oldValue: GModel) => GModel)) => void;
  • state: 可是是一个 atom 或者是一个 compute.
